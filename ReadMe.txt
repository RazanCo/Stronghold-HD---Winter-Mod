Stronghold HD - Winter Mod zip folder contents:
    Backup gm files - backup of the original gm files Stronghold HD.
    Christmas gm files - winter version of gm files for Stronghold HD.
    Read me - this.

To Install:

    - extract gm folder from "Christmas gm files" file to Stronghold HD install location on your system.
    - replace all 180 files by clicking replace all.
    - game is ready for playing.

To Uninstall:
    - open Backup gm files" folder and extract "gm" folder from within to Stronghold HD install location on your system.
    - replace all 180 files by clicking replace all.
    - game is ready for playing.
